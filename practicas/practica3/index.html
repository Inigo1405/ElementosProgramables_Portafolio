Iñigo Quintana Delgadillo
Ingeniería en Sistemas Computacionales
Fecha: Agosto 30, 2023
Robótica Arduino Attiny85

- Resumen -

Para la tercera práctica se realizarón 14 circuitos, se utilizo el Arduino UNO como programador de Attiny85 que es el microcontrolador para los siguientes trabajos.

- Introducción -

El ATtiny85 es un microcontrolador AVR de 8 bits de alta eficiencia y bajo consumo de energía, con periféricos como temporizadores y PWM, que se destaca por su versatilidad y tamaño compacto en un encapsulado de 8 pines, siendo ampliamente utilizado en proyectos de electrónica y sistemas embebidos para aplicaciones que van desde control de luces LED hasta sistemas de monitoreo y control, siendo programable mediante herramientas como AVR-GCC y programadores de microcontroladores.

- Materiales -

  • Leds
  • Resistencias 220 ohms
  • Arduino UNO
  • Attiny85
  • Botón
  • Protoboard

- Desarrollo -

Para ralizar esta práctica se siguieron los pasos a continuación:

  • Instalar librería Attiny
  • Arduino as ISP
  • Configurar tarjeta Attiny
  • Quemar bootloader
  • Probar código de ejemplo

Esto para configurar el Arduino UNO como programador y cambiar el estado inicial del Attiny. Sin estos pasos el microcontrolador Attiny no se podría programar, por lo tanto no se realizarían los ciercuitos a continuación.

- Resultados -

Una vez que se realizo el proceso de quemado del Attiny85, cada que se le programa algo este lo mantendra, por lo que si vuelve a conectarse en otro lado volvera a ejecutar lo anteriormente programado.

- Conclusiones -

Con la ventaja de que el Attiny85 maniente lo que se le programo, este puede volver a ser utilizado sin necesidad de repetir el conectado y progamado de este microcontrolador.
